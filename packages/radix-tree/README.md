# @abernier/radix-tree

A radix tree component.

## Installation

```bash
npm install @abernier/radix-tree
# or
pnpm add @abernier/radix-tree
# or
yarn add @abernier/radix-tree
```

## Usage

See the [Storybook documentation](https://main--697b1f722390b05fb8c0029c.chromatic.com) for usage examples.

## Development

This package is part of the [radix-primitives](../../README.md) monorepo.

To work on this package:

```bash
# Install dependencies from the root
pnpm install

# Build this package
pnpm --filter @abernier/radix-tree build

# Run tests
pnpm --filter @abernier/radix-tree test

# Run Storybook (from root, shows all packages)
pnpm run storybook
```
